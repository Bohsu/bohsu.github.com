####jekyll使用
1、jekyll安装

    $ gem install jekyll

2、启动服务

    $ jekyll serve   
    # 本地启动jeklly，默认4000端口
	 
3、 创建一个帖子

    $ rake post title="Hello World"  
    # 新建名为“Hello World”的文章（如有同名，不会覆盖）

4、创建一个页面
     
    $ rake page name="about.md"

jekyll使用参考:[http://jekyllbootstrap.com/](http://jekyllbootstrap.com/usage/jekyll-quick-start.html)
